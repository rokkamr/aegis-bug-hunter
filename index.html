Security & Privacy Guarantee

Aegis is built on a client-side sandbox architecture.

  • Local Processing: All folder directory exploration, tree reading, and file editing are done directly in your browser using standard HTML5 APIs.
  • Secure Transport: Your code is sent directly from your browser to Google's API servers. No intermediary server stores or processes your code.
  • Encrypted Storage: Your Gemini API Key is stored only in your local browser's localStorage, which remains on your machine and cannot be accessed by other websites.
